<h1 align="center"> Projeto Integrador - 2ºSemestre - Poligera 🧠</h1>
Projeto de extensão, realizado pelos alunos do Instituto Mauá de Tecnologia em parceria com o colégio Poliedro. Consiste na criação de uma plataforma web para geração de imagens utilizando inteligência artificial, direcionado à utilização pedagógica nas áreas de física e química. 

# 📌Público-alvo
> Alunos do Ensino Médio(15 a 18 anos) e professores do colégio Poliedro

# Status do projeto
> ✔️ Projeto Finalizado!

# :hammer: Funcionalidades do projeto
- `Funcionalidade 1`: Cadastro de aluno e professor.
- `Funcionalidade 2`: Sistema de autenticação.
- `Funcionalidade 3`: Geração de imagens.
- `Funcionalidade 4`: Envio de prompts em linguagem natural.
- `Funcionalidade 5`: Seleção de opções padronizadas.
- `Funcionalidade 6`: Edição de imagens.
- `Funcionalidade 7`: Exportação de imagens.
- `Funcionalidade 8`: Visualização de histórico.
- `Funcionalidade 9`: Sistema responsivo.

# :computer: Tecnologias utilizadas 
- `HTML`
- `CSS`
- `Javascript`
- `MongoDB`
- `Python`
